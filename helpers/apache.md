# Apache


Configurations must have the `.conf` extension name.

## Sites

To enable or disable a site.

```bash
a2ensite {site_conf}
```
```bash
a2dissite {site_conf}
```

## Modules

To enable modules.

```bash
a2enmod {mod_conf}
```
```bash
a2dismod {mod_conf}
```

## Configurations

To enable configurations.

```bash
a2enconf {mod_conf}
```
```bash
a2disconf {mod_conf}
```
